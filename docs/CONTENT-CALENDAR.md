## Content Creation Process

1. **Keyword research** → Identify target keyword + secondary keywords
2. **SERP analysis** → Check current top 10, identify content gaps
3. **Outline** → H1, H2s, H3s, FAQ questions, internal links
4. **Write** → 1,500+ words, include at least 1 original image/diagram
5. **Schema** → Add Article + HowTo/FAQPage as appropriate
6. **Metadata** → Unique title, meta description, OG tags
7. **Internal links** → Link to at least 2 existing pages + 1 tool page
8. **Publish** → Add to sitemap, submit URL to GSC
9. **Monitor** → Track rankings after 2-4 weeks
